# CodeAlpha Phishing Awareness Training

## 📌 Project Overview
A cybersecurity awareness training project designed to educate users on phishing attacks, how they work, how to identify them, and practical steps to prevent falling victim. This project combines an educational PowerPoint presentation with an interactive quiz to reinforce learning.

## 🎯 Training Objectives
This training helps users to:

- Understand what phishing is
- Learn the different types of phishing attacks
- Understand how phishing attack chains work
- Identify suspicious emails and fake websites
- Recognize social engineering tactics
- Learn phishing prevention best practices
- Test knowledge through an interactive quiz

## 📚 Topics Covered
- What is Phishing?
- Types of Phishing Attacks
- How Phishing Works (Attack Chain)
- Recognizing Phishing Emails
- Detecting Fake Websites
- Social Engineering Tactics
- Real-World Phishing Incidents
- Best Practices & Prevention
